## Key Elements

### 1. Header
- **Logo**: Text-based "Newark AI Community" (left)
- **Navigation**: Home, Events, Blog, Get Involved, About (right on desktop)
- **Mobile**: Hamburger menu (â˜°) that opens full-screen overlay

### 2. Hero Section
- **7/5 column split** on desktop (content/sidebar)
- **Beginner-friendly banner** prominently placed
- **Dual CTAs**: Primary (red) and secondary (outline)
- **"What You'll Get" card** with checklist format

### 3. Events Preview
- **3 event cards** in 4-column layout (desktop)
- **Stacked full-width** on mobile
- **Visual tags** for location type (in-person/online/hybrid)
- **Red CTA buttons** on each card

### 4. Blog Preview
- **3 blog cards** matching event layout
- **Date prominent** at top of each card
- **Text links** instead of buttons (secondary importance)

### 5. Community CTA
- **Centered content** on light gray background
- **Dual buttons**: Discord (primary) and Events (secondary)
- **Short, friendly copy**

### 6. Footer
- **4-column layout** on desktop
- **Stacked columns** on mobile
- **Social icons** in contact section
- **Copyright and privacy link** at bottom

## Design Principles Applied

1. **Swiss Grid**: 12-column system, mathematical spacing
2. **Hierarchy**: Size and weight (not color except red accent)
3. **White Space**: Generous padding and margins
4. **Accessibility**: Large touch targets (44px min), semantic structure
5. **Mobile-First**: Content stacks naturally, full-width CTAs
6. **Progressive Disclosure**: Most important content first (events)

---

## Interactions

### Desktop
- **Hover states**: Buttons darken, links change to red
- **Focus states**: Red outline (2px) on keyboard navigation
- **Smooth scrolling**: Anchor links scroll smoothly

### Mobile
- **Tap targets**: Minimum 44x44px (buttons are 48px height)
- **Menu**: Slides in from right with overlay
- **Cards**: Full-width tap area
- **Forms**: Inputs expand to full width

---

## Notes for Development

- Use `swiss-grid` class for 12-column layout
- Event cards use `event-card` class
- Blog cards use `swiss-card` class
- Section labels use `swiss-section-label` class
- Buttons follow `.btn-swiss-red` and `.btn-swiss-outline` patterns
- Mobile breakpoint: 768px
