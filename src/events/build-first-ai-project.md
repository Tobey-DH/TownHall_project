## What You'll Build (In 2 Hours)

**Choose Your Project Path:**

### Option 1: AI Chatbot (Beginner-Friendly)
- Build a chatbot that answers questions about your favorite topic
- **Tools:** ChatGPT API (free tier) + simple template we provide
- **Skill Level:** No coding required (copy-paste friendly)
- **College App Impact:** Shows initiative and tech interest

### Option 2: Smart Recommendation System (Intermediate)
- Create a movie/book/music recommender using AI
- **Tools:** Python + free AI library (we'll set it up for you)
- **Skill Level:** Basic coding helpful but not required
- **College App Impact:** Demonstrates problem-solving and technical skills

### Option 3: AI Content Creator (Creative Students)
- Build a tool that generates art, poems, or stories using AI
- **Tools:** Hugging Face models (free) + our template
- **Skill Level:** No coding required
- **College App Impact:** Perfect for arts/humanities students
