## 4. Data Storage & Security

### 4.1 Where Your Data is Stored

**Contact & Event Information**:
- Stored in HubSpot CRM (US-based)
- Encrypted in transit and at rest
- Backed up daily

**Analytics Data**:
- Stored by Plausible Analytics (EU-based)
- Never transferred to US servers
- Compliant with EU data residency laws

**Website Content**:
- Hosted on GitHub Pages (US-based)
- Content delivered via CDN

### 4.2 Data Retention

**Contact Information**:
- Retained for 3 years unless you request deletion
- Deleted automatically after inactivity period
- Can be deleted upon request

**Analytics Data**:
- Retained for 30 days (default)
- Older data automatically deleted
- No long-term user history maintained

**Event Attendance Records**:
- Retained for 2 years for legal/tax purposes
- Deleted upon your request

**Newsletter Subscribers**:
- Retained until you unsubscribe
- Can unsubscribe anytime via email

### 4.3 Security Measures

We implement:
- ✅ HTTPS encryption for all pages
- ✅ Secure form submissions
- ✅ Regular security updates
- ✅ Access controls (staff only)
- ✅ Password protection for admin areas
- ✅ Regular backups
- ✅ Security monitoring

**However**: No method of transmission over the Internet is 100% secure. We cannot guarantee absolute security.

### 7.2 GDPR Compliance Measures
